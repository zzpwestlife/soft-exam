# 第一部分：考试概览

## 1.1 考试基本信息

| 项目 | 内容 |
|------|------|
| 考试名称 | 计算机技术与软件专业技术资格（水平）考试 - 系统架构设计师 |
| 级别 | 高级 |
| 考试形式 | 机考（2023年起全面机考） |
| 考试频率 | 每年两次（5月、11月） |

## 1.2 考试科目与时间

| 科目 | 考试时间 | 题型 | 分值 |
|------|----------|------|------|
| **科目一：信息系统综合知识** | 150分钟 | 单项选择题（75道） | 75分（45分合格） |
| **科目二：系统架构设计案例分析** | 90分钟 | 问答题（1必做 + 4选2） | 75分（45分合格） |
| **科目三：系统架构设计论文** | 120分钟 | 论文题（4选1） | 75分（45分合格） |

> ⚠️ 三科须**一次同时通过**，单科成绩不保留。

## 1.3 官方考纲知识体系

### 科目一：信息系统综合知识（选择题）
1. **计算机软件与网络基础知识** — 操作系统、数据库系统、嵌入式系统、计算机网络、多媒体、系统配置与性能评价
2. **信息化基础知识** — 信息系统工程规划、电子政务、企业信息化与电子商务、信息资源管理
3. **系统开发基础知识** — 开发管理、需求管理、软件开发方法、设计方法、基于构件开发、测试与评审
4. **软件架构基础知识** — 架构概念、架构风格、特定领域软件架构、基于架构的开发方法、架构评估、软件产品线
5. **安全性与可靠性技术** — 信息安全与保密、系统可靠性、安全法规
6. **标准化与知识产权** — 标准化意识、各类标准、知识产权
7. **应用数据（数学）** — 概率统计、图论、组合分析、运筹方法、数学建模
8. **专业英语** — 高级工程师级别英文阅读能力

### 科目二：系统架构设计案例分析（问答题）
1. 系统规划
2. 软件架构设计（含XML技术、质量属性、架构风格、架构评估、软件产品线）
3. 设计模式（含概念、组成、分类、实现）
4. 系统设计（含处理流程、人机界面、数据库、网络系统、中间件、性能设计）
5. 软件系统建模（含结构化建模、面向对象建模、UML、数据库建模）
6. 分布式系统设计（含通信协议、基于对象/Web/消息的分布式设计、异构互操作）
7. 嵌入式系统设计（含实时系统、任务调度、中断处理）
8. 系统可靠性分析与设计（含故障模型、可靠度计算、备份恢复）
9. 系统安全性和保密性设计（含访问控制、数据加密、通信安全）

### 科目三：系统架构设计论文（4选1）
1. 系统建模
2. 软件架构设计
3. 系统设计
4. 分布式系统设计
5. 系统可靠性分析与设计
6. 系统安全性和保密性设计
